# cURL examples

One-liner samples. Export your API key once:

```bash
export TEXTORY_API_KEY=sk_live_xxxxxxxxxxxxxxxxxxxxxx
```

Then run:

```bash
bash send-phone.sh         # global SMS via paired phone
bash send-web.sh           # Korea-only web gateway (requires TEXTORY_SENDER)
bash get-balance.sh        # credit balance
bash list-senders.sh       # registered sender numbers
```
